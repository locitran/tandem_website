Overview

The TANDEM-DIMPLE web server provides two core functions:

  • Inferencing: Predicting the pathogenicity of single amino acid variants (SAVs) using pre-trained models.
  • Training: Training gene/disease-specific models using user-curated variant datasets to improve prediction accuracy from gene/disease-general models.

Below is a step-by-step guide to using the webserver.

Step 1: Create a new session or resume a previous one by entering a session ID.

The session ID functions like a user ID. Under one session ID, you can manage several jobs.

Step 2: Configure and Submit a Job

Select a mode: “Inferencing” or “Training” (as defined above). Enter SAV queries in the text field, separated by new lines. For “Training” mode, each SAV must include a label, 1 for pathogenic and 0 for benign.

Select an example and click “Load Input” to fill the fields. Click “View output” to open the results. Click “Clear all” to reset the inputs.

Select a model from the dropdown menu (Inferencing only). Available models include TANDEM, TANDEM-DIMPLE for GJB2, and TANDEM-DIMPLE for RYR1. Models trained in previous jobs will also appear in the list.

Provide a PDB or AlphaFold2 ID or upload a coordinate file for SAV feature generation (optional). If no structure is provided, the system automatically selects a representative structure from UniProt based on sequence coverage, resolution, and deposition date.

Assign a job name and submit your job!

Step 3: Monitor Job Status (This section is under revision)
  • Submission Status: Displays all submitted parameters or reports errors.
  • Processing Status: Shows real-time execution progress.
  • Click New Job to start a new job.

Pathogenicity probabilities and binary classifications are summarized for queried SAVs spanning multiple proteins (UniProt IDs O00189, O00194, O00204, and O00206). Variants are classified as pathogenic if the predicted probability is ≥ 0.5 and benign if the probability is < 0.5. The Performance of TANDEM-DIMPLE Model of PKD1/2 on PKD Pathogenicity Predictions

To demonstrate the training utility of the website, we use clinically validated SAVs from the proteins Polycystin-1 and Polycystin-2 (PC1 and PC2, also known as PKD1 and PKD2), which are etiologically linked to autosomal dominant polycystic kidney disease (ADPKD). In the human body, these two proteins form a heteromeric receptor-channel complex essential for mechanosensitive calcium signaling in renal primary cilia. The dataset comprises 153 pathogenic and 101 benign SAVs collected from a previous study (Yu et al., 2022) and from two public databases (ClinVar and LOVD). Here, we use these SAVs to fine-tune the gene-general TANDEM model and examine the improvement before and after training with the TANDEM-DIMPLE web server.

Input:

  • SAVs: PKD1-PKD2-SAVs-with-labels.txt
  • PKD1/2 structure: PKD1-PKD2-opm-complex.pdb

Output: https://dyn.life.nthu.edu.tw/TANDEM/results/?session_id=test&job_name=PKD_demo

Following the same protocol applied to GJB2 and RYR1, variants were partitioned into training sets (30%), validation sets (60%; training stopped when validation loss no longer declined), and a hold-out test set (10%), generating 15 PKD1/2-specific TANDEM-DIMPLE models. As a default, our TANDEM website shows both gene-general and gene-specific results for a gene-specific test set, where the gene-general model may still be recommended if it outperforms a gene-specific model.
